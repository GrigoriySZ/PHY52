1) Позволяет локально развернуть React проект
npm create vite@latest test 

// сразу создаст 
npm create vite@latest team-dashboard -- --template react

2) Перейти в папаку проекта
cd nameProject

3) Скачивание всех нужных библиотек
npm install

// Библиотека React с иконками
npm install react-icons

4) Запуск проекта на локальном сервере в режиме разработки
npm run dev

5) // Команда для остановки сервера 
[ctrl+c] внутри проекта
